HiMaC 2026 Events Website
A modern, responsive website for the Hindusthan Management Carnival (HiMaC) 2026 Events.

Features
Event Grid Display: Beautiful card-based layout showcasing all events
Event Categories: Events organized by category (Technical, Cultural, Entrepreneurship, Fun)
Interactive Modals: Click on any event card to view detailed information
About Section: Information about HiCET and HiMaC
Responsive Design: Works seamlessly on desktop, tablet, and mobile devices
Modern UI: Clean, colorful design with smooth animations

How to Use
Open index.html in a web browser
Browse events in the grid layout
Click on any event card to view detailed information
Click "About" in the navigation to learn more about HiCET and HiMaC
Click "Register Now" buttons to register (registration functionality can be added)
