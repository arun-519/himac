# Event Images

Place the following images in this directory:

1. **bi-battle.jpg** - Analytics/Dashboard image showing data visualizations and KPIs
2. **stock-war.jpg** - Financial candlestick chart with trading data
3. **best-manager.jpg** - People working collaboratively on laptops in a workspace
4. **talent-show.jpg** - Concert/festival stage with colorful lights and crowd
5. **mime.jpg** - Silhouettes of performers against a red curtain backdrop
6. **ipl-auction.jpg** - TATA IPL AUCTION logo with blue background and colorful design elements
7. **face-painting.jpg** - Abstract face painting effect with neon colors on black background
8. **startup-expo.jpg** - Modern office space with industrial design, desks, and collaborative workspace

## Image Specifications

- **Format**: JPG or PNG
- **Recommended Size**: 600x400 pixels (or similar aspect ratio)
- **Aspect Ratio**: 3:2 or 16:9
- **File Size**: Optimize for web (under 500KB each)

The images will automatically scale to fit the event cards while maintaining their aspect ratio.
